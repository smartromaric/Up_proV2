Courses & Partenaires Intégration

Endpoints pour la liste des courses et les partenaires.

Liste des Courses
GET /v1/franchises/{franchiseId}/orders?page=1&limit=25
Authorization: Bearer {token}

Récupère la liste paginée des courses pour une franchise donnée. Fonctionnel ✓

Problème identifié : Filtre par Service
Le frontend envoie le paramètre service (taxi, delivery, rental, freight) mais le backend semble ne pas le prendre en compte.
Action requise : Vérifier avec le backend le nom exact du paramètre (service vs service_type ?)
Détail d'une Course
GET /v1/admin/orders/{orderId}
Authorization: Bearer {token}

Récupère les détails complets d'une course. Fonctionnel ✓

Liste des Partenaires
GET /v1/franchise/partners?page=1&per_page=5
Authorization: Bearer {token}

Utilisé pour afficher les partenaires récents sur le dashboard. Fonctionnel ✓

Contexte Franchise
GET /v1/franchises/me
Authorization: Bearer {token}

Récupère l'ID de la franchise connectée pour les appels API. Fonctionnel ✓
